# Hitbox Motion Training Tracker

A system that analyzes hitbox directional inputs and tracks motion execution such as QCF and DP.

## Features
- Input logging
- Motion detection
- Training session tracking
- SQL analytics

## Tech Stack
- Python
- SQL Server
- Hitbox Controller (Tachyon Black Plus PCB)
